# 项目介绍

- 项目目的：实践常见后台管理系统功能
- 该项目分为前端后端两部分，此为前端部分，后端部分请查看xxx
- 项目技术：vite + react19 + router7 + redux + antd + axios
- 功能包含：
  - [ ] 用户管理
  - [ ] 角色管理
  - [ ] 权限管理
  - [ ] 日志管理
  - [ ] 文件管理
  - [ ] 数据统计图表展示
  - [ ] 内容管理：用于打通前后端流程，内容主要包含部分唐诗宋词
  - [ ] 富文本编辑/预览
  - [ ] 系统管理
  - [ ] 视频操作
  - [ ] 地图定位导航

## 运行项目

1. 安装依赖包：pnpm install
2. 启动项目：pnpm run dev
3. 访问地址：<http://localhost:3000>
